# Workhours

This api helps organize workhours of employees

## Framework
Build using [Lumen](https://github.com/laravel/lumen) on top of PHP 7.4

## Requirements
- Bash
- Docker
- Docker compose

_Linux systems are prefered_

## Architecture 
- Docker containers
  - PHP 7.4
  - nginx (server)
  - mysql 5.7
